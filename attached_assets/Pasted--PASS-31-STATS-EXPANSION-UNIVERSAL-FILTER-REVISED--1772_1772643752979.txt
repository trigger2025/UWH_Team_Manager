---------------------------------------
PLAYER STAT CALCULATIONS
---------------------------------------

For each player (from finalised results only):

gamesPlayed
wins
losses
draws

winRate =
    wins / gamesPlayed (if > 0)

totalRatingChange =
    sum of ratingDelta from finalised matches

currentRating =
    player.rating (already stored)

ratingTrend =
    currentRating minus rating
    from 10 matches ago
    (or first recorded match if <10)

---------------------------------------
NEW: STREAK TRACKING
---------------------------------------

For each player:

currentWinStreak =
    consecutive wins counting backward from most recent result

currentLossStreak =
    consecutive losses counting backward

bestWinStreak =
    highest consecutive wins across history

worstLossStreak =
    highest consecutive losses across history

---------------------------------------
MAIN STATS TABLE (IN STATS TAB)
---------------------------------------

Columns:

Name
Current Rating
Games
W
D
L
Win %
Net Rating Δ
Current Streak

Streak Display:
W3 = 3 win streak
L2 = 2 loss streak
- = none

Clicking player row expands:

• Last 5 results summary
• Best win streak
• Worst loss streak
• Total A pool games
• Total B pool games
