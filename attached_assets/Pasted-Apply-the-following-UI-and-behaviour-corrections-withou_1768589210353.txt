- Remove the global “Use off-hand ratings” toggle.

  - 1-3-2: Forward, Wing, Centre, Back

- Formation choice must be saved in generated team state and match snapshots.

- Remove the tournament rating reduction toggle from the Match Results UI.

- Rating adjustments in tournament mode must be handled internally, not via user toggles.

- Add the ability to delete individual match results.
- Deleting a match result must NOT revert ratings automatically.

  - Deleting match results
